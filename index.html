Bootstrap Pager

Usage

Basic

Enable uloader via JavaScript:

$('#myPager').pager(options)

Options

Name type default description
pages int 0 The total pages
show string auto Hides the pager when pages <= 0
hasPrev boolean true Shows the previous button
hasNext boolean true Shows the next button
current int 1 The current page when initialized
continuous boolean false Shows previous/next group when previous/next page in previous/next group
size string middle The button size, can be small, middle, large
count int 8 The count of showing pages
onchange function {} Be called when page change

Methods

Show previous page

$('#myPager').pager('prevPage')

Show next page

$('#myPager').pager('nextPage')

Show previous group

$('#myPager').pager('prevGroup')

Show next group

$('#myPager').pager('nextGroup')

Events

Event Type Description
change.bs.pager This event fires immediately when the pages change
